# CarbonWise

CarbonWise is a modern web application designed to help users track, understand, and reduce their carbon footprint.

## Features

- **Carbon Footprint Calculator**: Log daily activities like transport, electricity, and diet.
- **Interactive Dashboard**: Visualize your emissions trends with dynamic charts (Chart.js).
- **Daily Eco Challenges**: Complete challenges to earn points and level up.
- **Impact Goals**: Set monthly reduction targets and track your progress.
- **Eco Education**: Daily sustainability facts and tips.
- **PDF Reports**: Export your impact data as a professional PDF report.
- **Dark Mode Support**: Seamlessly switch between light and dark themes.

## Tech Stack

- **Frontend**: HTML5, CSS3, ES6 JavaScript
- **Icons**: Lucide Icons
- **Charts**: Chart.js
- **PDF Generation**: html2pdf.js
- **Persistence**: Local Storage

## Getting Started

Simply open `index.html` in any modern web browser to start tracking your carbon footprint. All data is stored locally on your device.
